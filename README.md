La aplicación móvil fue desarrollada en Flutter y se comunica con un servidor Python desplegado en Railway mediante Sockets TCP puros. El servidor está ubicado en Estados Unidos y el cliente en Nicaragua, demostrando comunicación real entre dos equipos en redes completamente distintas y remotas. La app permite registro, inicio de sesión, visualización del catálogo, carrito de compras y generación de proforma en PDF enviada al correo del cliente.
